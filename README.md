# HISTORIA GLOBAL XIV–XVIII · Presentación HUD

Presentación interactiva estilo **HUD / sistema operativo de robot** sobre los Trabajos Prácticos 3 y 4 de Historia 2 (América y Europa entre los siglos XIV y XVIII — Santillana).

**Integrantes:** Aarón Villalba · Marcos Vilte
**Docente:** Profe Lucas Raffo Benegas

## Cómo verla

Abrí `index.html` en cualquier navegador moderno (Chrome, Edge, Firefox). No requiere instalación ni build: React, Tailwind y Framer Motion se cargan por CDN.

> Necesita conexión a internet la primera vez para descargar las librerías desde CDN.

## Controles

- `→` / `Espacio` / `Enter` → siguiente slide
- `←` → slide anterior
- `Home` / `End` → ir al inicio / final
- También podés usar las flechas en pantalla o los puntos del indicador.
